The Shopper's Card

The Shopper's Card is a convenient way to store all your loyalty cards in one by using your ID number to set it up. This website is the front-end design which allows a user to sign up, view the points on their card, customise the design and contact the company for additional help. The back-end is currently still in development. I plan to use PHP and MySQL to set up the database.

Project Difficulties

Learning how to make all web pages responsive.

Solution

Through the use of media queries I was able to make the website appealing across devices, however it is still under development so there are some features that look out of place and I am working towards further updates.

Amelgo UK

Amelgo UK is a luxury fragrance brand that initially had their e-commerce website built on Wordpress. My client requested that I migrate the website to Shopify and set it up to the original design. In order to complete this project I needed a strong work ethic as it tested both my soft and technical skills. Due to company policy I am unable to display the code on github.

Project Difficulties

Customizing a ready made template in order to meet client requirements within a tight deadline.

Solution

Time management was a major factor in this project. It was necessary for me to read and understand the already written code and adapt quickly using Liquid to make the website dynamic. I needed to spend my time wisely in order to research and communicate with the client throughout the development process.The website is still undergoing changes.
